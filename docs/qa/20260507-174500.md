### 要修正・要確認の内容

1. **CLIエクスポートQAが実行不可（重要）**
   - Mac App Store 向けサンドボックス有効化により、CLI 引数ファイルパスが読めなくなった
   - **対策案A**: Debug 専用エンタイトルメントファイル（`ActivityVideoStudio-Debug.entitlements`）を作成し、Debug ビルドでは `com.apple.security.app-sandbox = false` に戻す。App Store 配布用（Release）は現状維持
   - **対策案B**: テスト用ファイルをアプリのサンドボックスコンテナ（`~/Library/Containers/com.avs.ActivityVideoStudio/`）に配置する
   - **対策案C**: シナリオをGUI操作（Automator / AppleScript）に切り替える

2. **チャプターマーカーのエクスポート未対応（前回から継続）**
   - `ChapterMarker` が `VideoExporter` / `OverlayRenderer` に渡されていない
   - チャプターマーカーをエクスポートに反映するには `OverlayRenderer` に `chapterMarkers` プロパティを追加し、エクスポート前に `PreviewViewModel` からセットする必要がある

3. **`docs/scenarios.md` のパラメータ誤記（前回から継続）**
   - `--trim-end 29` は「末尾29秒カット」であり「最初29秒を出力」ではない
   - 短尺エクスポートには `--trim-end <total - 29>` が正しい
